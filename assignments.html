Assignment 2: KMeans Clustering Visualization

This assignment involves implementing the KMeans clustering algorithm from scratch and developing an interactive web application to visualize different initialization strategies (Random, Farthest First, KMeans++, and Manual). I structured the application with React for the UI and used JavaScript to implement the algorithm, carefully designing it to allow for dynamic updates at each step. The project was approached by first implementing the algorithm with a focus on efficiency and then adding various visualization modes to display how the clustering progresses over time. Assignment 3: SVD Preprocessing and Logistic Regression on MNIST

In this assignment, I applied Singular Value Decomposition (SVD) as a preprocessing technique on the MNIST dataset, followed by training a logistic regression model to classify the digits. I compared model performance and training time across different levels of dimensionality reduction by varying the number of SVD components. The results showed a clear trade-off between accuracy and training efficiency. Additionally, I visualized the top 5 singular vectors to gain insight into the features captured by SVD.

GitHub Repository: Github Link

Assignment 4: LSA Search Engine using 20 Newsgroups

This assignment involves developing a basic search engine that implements Latent Semantic Analysis (LSA) to retrieve documents based on user queries. The search engine uses the Scikit-learn version of the 20 Newsgroups dataset and ranks documents using cosine similarity. The project includes a web application that allows users to input queries and returns the top relevant documents. LSA was applied to reduce dimensionality and improve search relevance by capturing latent patterns in the document-term matrix. Demo Video: here.

Assignment 6: Linear Regression

For this assignment, I developed a Flask web application that allows users to explore how randomness impacts linear regression. By setting parameters like sample size, mean, variance, and the number of simulations, users can generate random data and see how regression lines vary in the absence of true relationships. The application includes interactive scatter plots and histograms that visualize the slope and intercept distributions from multiple simulations. Demo video can be found here.
